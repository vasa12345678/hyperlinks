Сервисы Интернет

Сервис Telnet

Сервис Telnet позволяет передавать информацию, которая вводится на вашей клавиатуре, другому компьютеру на обработку, а результаты отображать на вашем мониторе.

Сервис FTP

Сервис FTP позволяет получать и передавать файлы по сети. Установив связь с удаленным компьютером, пользователь может скопировать файл с удаленного компьютера на свой или скопировать файл со своего компьютера на удаленный.

Сервис E-mail

Электронная почта — обмен почтовыми сообщениями с любым абонентом сети Internet. Существует возможность отправки как текстовых, так и двоичных файлов. На размер почтового сообщения в сети Internet накладывается следующее ограничение — размер почтового сообщения не должен превышать 64 килобайт.

Задание №7


Словарь терминов

А Б В Г Д

А

Алгоритм - чётко определённая последовательность инструкций, предназначенная для решения конкретной задачи или достижения определённой цели.

Архитектура - общая структура компьютерной системы, определяющая её компоненты, взаимосвязи между ними и принципы функционирования.

API (интерфейс прикладного программирования) - набор правил и протоколов, позволяющих различным программным компонентам взаимодействовать друг с другом.

Абстракция - процесс выделения существенных характеристик объекта или системы и игнорирования несущественных деталей.

Адрес - Уникальный идентификатор местоположения данных в памяти компьютера или сети.

Б

База данных - организованная структура данных, предназначенная для хранения, управления и извлечения информации.

Бит - наименьшая единица информации в компьютере, принимающая одно из двух значений: 0 или 1.

Браузер - программное обеспечение для просмотра веб-страниц и доступа к интернет-ресурсам.

Бэкап (Backup) - Резервная копия данных, предназначенная для восстановления информации в случае её потери или повреждения.

Булева алгебра - Раздел математики, имеющий дело с логическими операциями и булевыми переменными (принимающими значения “истина” или “ложь”).

В

Виртуализация - Технология, позволяющая создавать виртуальные версии физических ресурсов (например, серверов, операционных систем).

Вектор - Одномерный массив данных, часто используемый в математических и научных вычислениях.

Вирус - Одномерный массив данных, часто используемый в математических и научных вычислениях.

Веб-сервер - Программа, предоставляющая доступ к веб-сайтам и веб-приложениям через интернет.

Вычислительная сеть - Совокупность компьютеров и других устройств, соединенных между собой для обмена данными и совместного использования ресурсов.

Г

Граф - Абстрактная структура данных, представляющая собой набор объектов (вершин) и связей между ними (рёбер).

GUI (Graphical User Interface) - Графический интерфейс пользователя, позволяющий взаимодействовать с компьютером посредством визуальных элементов (кнопок, значков, окон).

Глобальная переменная - Переменная, доступная для использования во всех частях программы.

Генератор случайных чисел - Алгоритм или устройство, создающее последовательность чисел, которая кажется случайной.

Гипертекст - Текст, содержащий ссылки на другие тексты или ресурсы.

Д

Данные - Информация, представленная в форме, пригодной для обработки компьютером.

Дебаггинг (Debugging) - Процесс поиска и устранения ошибок в программном коде.

Диск - Устройство для хранения данных, например, жёсткий диск (HDD) или твердотельный накопитель (SSD).

Драйвер - Программа, обеспечивающая взаимодействие операционной системы с аппаратным обеспечением.

Динамическая память - Область памяти, выделяемая во время выполнения программы по мере необходимости.
